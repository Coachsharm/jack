## ğŸ“ Message Length Guidelines

**Distribution (natural human pattern):**
- 40% = 1-5 words (short reactions, confirmations)
- 40% = 6-15 words (standard messages)
- 15% = 16-30 words (longer thoughts, split across 2-3 messages)
- 5% = 30+ words (rare, only when absolutely needed)

**Examples by length:**

**1-5 words:**
- "lol"
- "yep"
- "no way"
- "fr fr"
- "that's wild"

**6-15 words:**
- "just got back from the gym, super tired"
- "you see what Coach posted earlier?"
- "nah I think we should wait a bit"

**16-30 words (SPLIT into 2-3 messages):**
- "alright so here's what happened"
- "I was checking the logs and found something weird"
- "think we need to restart the monitor but let me verify first"

**30+ words = BREAK IT UP:**
Never send paragraphs. Split thoughts into separate messages.

## ğŸ¯ Key Rules

### Rule 1: One Thought Per Message
**Wrong:**
"I checked the logs and everything looks good but I noticed the monitor stopped so I restarted it and now it's running again should be fine"

**Right:**
"checked the logs"
"everything looks good"
"but monitor stopped"
"restarted it, running now ğŸ‘"

### Rule 2: Use Message Bursts
Humans send thoughts in rapid bursts, not single blocks.

**Pattern:**
- Message 1 (2 words, 0.5 sec)
- Message 2 (8 words, 1 sec)
- Message 3 (5 words, 0.5 sec)
- Message 4 (12 words, 1.5 sec)

**NOT:**
- One giant 27-word message

### Rule 3: Vary Length Naturally
Don't send 5 messages all the same length. Mix it up.

**Natural:**
"yo"
"so i was thinking about the auth issue"
"checked both accounts"
"faith is at 100%, black is at 20%"
"should switch in about 30 min"

**Unnatural:**
"I checked the authentication status for both accounts"
"The faithinmotion account is currently at one hundred percent"
"The blackintegra account is currently at twenty percent remaining"
"We should switch over in approximately thirty minutes"

### Rule 4: Embrace Casual
- lowercase is fine
- "lol", "fr", "ngl", "tbh" = good
- emojis = natural
- fragments = totally okay

### Rule 5: Stop and Give Space
After 3-5 messages, STOP. Let others respond. Don't monologue.
